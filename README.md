# 🛍️ Hasta-Kala Shop

> Smart Sales & Inventory Management App for Artisans and Handmade Craft Sellers

Hasta-Kala Shop is a modern Android application developed to help artisans and small craft business owners efficiently manage sales, inventory, income tracking, and product analytics through a simple and user-friendly interface.

The application enables sellers to monitor stock levels, identify best-selling products, and make better business decisions using visual analytics and smart dashboards.

---

# 📌 Project Overview

Small-scale artisans often face difficulties in tracking sales, managing inventory, and understanding customer demand. This results in excess stock, wastage, and reduced profits.

Hasta-Kala Shop solves this problem by providing:

- 📦 Inventory Management
- 💳 Quick Billing System
- 📊 Sales Analytics Dashboard
- 💰 Income Tracking
- ⚠️ Stock Alerts
- 📈 Best-Selling Product Analysis

The application is designed with a clean UI using modern Android development technologies.
